  97. def toLocal: Word2VecModel

    Returns a local Word2VecModel, the default Spark implementation.

    Returns a local Word2VecModel, the default Spark implementation. This can be used if only the training should be performed with a Glint cluster.

    Note that this implementation pulls the whole distributed matrix to the client and might therefore not work with large matrices which do not fit into the client's memory.

    Note also that while this implementation can train large models, the word vectors in the default Spark implementation are limited to 8GB because of the broadcast size limit.
