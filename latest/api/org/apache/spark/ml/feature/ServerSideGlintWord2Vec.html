Related Docs: object ServerSideGlintWord2Vec | package feature

final class ServerSideGlintWord2Vec extends Estimator[ServerSideGlintWord2VecModel] with ServerSideGlintWord2VecBase with DefaultParamsWritable

ServerSideGlintWord2Vec trains a model of Map(String, Vector), i.e. transforms a word into a code for further natural language processing or machine learning process.
